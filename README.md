## 1. Fontis Fine Dine  

**Files Included:**  
- fontis_fine_dine_App-master.zip – Android Application  
- Fontis-Fine-Dine-Web-main.zip – Web Application  

### Project Overview  
Fontis Fine Dine is a fast-food restaurant specializing in Kota, a popular South African street food.

This project includes both a website and an Android application designed to:

- Allow customers to place orders online  
- Track order progress  
- Complete payments efficiently  
- Improve customer convenience  
- Modernize the restaurant ordering experience  

### Objective  
To design and develop a simple, user-friendly digital ordering system that enhances accessibility while maintaining operational efficiency.
